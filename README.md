# EnglishQuest

EnglishQuest — 面向FCE考试的趣味英语学习平台

## 技术栈

- React 18 + TypeScript
- Vite
- Tailwind CSS
- React Router
- Recharts
- Framer Motion
- Lucide React

## 快速开始

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 功能模块

- 入学能力测评
- 个性化学习计划
- 词汇冒险
- 语法挑战
- 阅读探索
- 听力训练
- FCE考试中心
- 游戏化系统（等级、连胜、排行榜、成就、商店）
- 个人中心
