## 1. 产品概述

### 1.1 产品定位

EnglishQuest 是一款以**通过剑桥 B2 First（FCE）考试**为终极目标的在线英语学习网站，采用**游戏化学习 + 个性化学习计划 + 考试导向训练**三位一体的产品策略，让用户在沉浸式的"冒险旅程"中每天坚持学习，最终达成 FCE 考试通过（160分+）的目标。

### 1.2 产品愿景

> 让每一个英语学习者都能像玩游戏一样期待每天的学习，在不知不觉中掌握通过 FCE 所需的全部能力。

### 1.3 核心价值主张

| 维度 | 传统学习平台 | EnglishQuest |
|------|------------|-------------|
| 学习动力 | 靠自律，容易放弃 | 游戏化机制驱动，"停不下来" |
| 学习路径 | 通用内容，千人一面 | 个性化学习计划，动态调整 |
| 目标导向 | 目标模糊 | 明确对标 FCE 考试能力要求 |
| 反馈机制 | 延迟反馈，缺乏成就感 | 即时反馈，可视化进步 |

## 2. 背景与目标

### 2.1 市场背景

- **FCE 考试需求增长**：FCE（B2 First）是剑桥英语五级证书考试的第三级，对应 CEFR B2 级别，证书全球永久有效。词汇量要求约 6000 词，对标雅思 5.5–6.5，是新高考的对标水平，也是衔接雅思与海外留学的关键节点。
- **学习坚持难**：语言学习最大的痛点不是"学不会"，而是"坚持不下去"。研究表明，超过 70% 的自学者在开始学习后的 3 个月内放弃。
- **游戏化验证有效**：Duolingo 通过连胜打卡（Streaks）、排行榜、奖励系统等游戏化设计，成功让数亿用户养成每日学习习惯，证明了"趣味性"对学习坚持的关键作用。

### 2.2 FCE 考试结构概览

本产品所有学习模块的设计均对标 FCE 考试的四大组成部分：

| 考试模块 | 时长 | 题量 | 占比 | 核心能力 |
|---------|------|------|------|---------|
| 阅读与语言运用 | 75 分钟 | 7 部分 52 题 | 40% | 阅读理解、词汇辨析、语法应用 |
| 写作 | 80 分钟 | 2 篇 | 20% | 议论文 + 多体裁写作 |
| 听力 | 约 40 分钟 | 4 部分 30 题 | 20% | 理解对话、独白、广播等 |
| 口语 | 13–15 分钟/组 | 4 个环节 | 20% | 个人陈述、双人讨论、协作任务 |

**评分体系**：满分 190 分；**160 分通过（Grade C）**，173 分优秀（Grade B），180 分卓越（Grade A）。FCE 设有"小分均衡线"——四项技能各有门槛，偏科将严重拖累总分，**备考必须四项均衡推进**。

### 2.3 产品目标

| 目标层级 | 具体目标 | 衡量指标 |
|---------|---------|---------|
| **用户目标** | 帮助用户通过 FCE 考试 | 用户模考平均分 ≥ 160；考试通过率 ≥ 75% |
| **行为目标** | 让用户每天坚持学习 | 日均学习时长 ≥ 20 分钟；7 日留存率 ≥ 60%；30 日留存率 ≥ 40% |
| **体验目标** | 学习过程有趣、有成就感 | 用户满意度 ≥ 4.5/5；日均使用频次 ≥ 2 次 |
| **商业目标** | 实现可持续增长 | 月活用户（MAU）持续增长；付费转化率 ≥ 8% |

## 5. 详细功能需求

### 5.1 学习计划系统（P0）

#### 5.1.1 入学能力测评

| 需求项 | 描述 |
|--------|------|
| 功能目标 | 评估用户当前英语水平，为生成个性化学习计划提供依据 |
| 测评内容 | 涵盖词汇、语法、阅读、听力四个维度的自适应测试 |
| 题目数量 | 自适应出题，约 40–60 题 |
| 测评时长 | 20–30 分钟 |
| 输出结果 | CEFR 等级定位 + 四项能力雷达图 + 与 FCE 要求的差距分析 |
| 技术要求 | 采用自适应测试算法（IRT），根据答题正确率动态调整题目难度 |

**测评结果展示示例：**

```
你的当前水平：CEFR B1（中级）

能力雷达图：
        词汇
         70%
       /      \
  语法 60%    阅读 75%
       \      /
        听力 55%

距 FCE 通过线差距：
- 词汇：需掌握约 2500 个新词
- 语法：需强化虚拟语气、倒装句等高阶语法
- 阅读：已接近 FCE 要求，需提升阅读速度
- 听力：差距较大，需重点突破
```

#### 5.1.2 个性化学习路径生成

| 需求项 | 描述 |
|--------|------|
| 功能目标 | 根据测评结果和考试日期，自动生成个性化学习计划 |
| 输入参数 | 当前水平、目标考试日期、每日可学习时长、薄弱项偏好 |
| 计划周期 | 分为三个阶段（详见下文） |
| 输出形式 | 可视化学习路径地图 + 每日任务清单 |
| 调整机制 | 每周根据学习数据自动微调计划 |

**三阶段学习计划设计：**

| 阶段 | 名称 | 时长占比 | 核心任务 | 阶段目标 |
|------|------|---------|---------|---------|
| 第一阶段 | 能力筑基 | 40% | 词汇积累、语法系统化、阅读输入、听力泛听 | 词汇量达到 5000+，语法体系完整，各项能力达 B1+ |
| 第二阶段 | 专项强化 | 35% | FCE 各题型专项训练、写作模板、口语话术 | 熟悉所有 FCE 题型，四项能力均衡达 B2 |
| 第三阶段 | 冲刺模考 | 25% | 全真模考、错题复盘、查漏补缺、考试策略 | 模考稳定 160+ 分，具备考试状态 |

#### 5.1.3 FCE 倒计时与里程碑

- 用户设置目标考试日期后，首页显示倒计时
- 学习路径上设置阶段性里程碑（每完成一个阶段解锁新区域）
- 里程碑节点自动安排阶段测评，检验阶段成果

### 5.2 每日学习模块（P0）

> **设计理念**：每个学习模块都以"关卡"形式呈现，将 FCE 考试所需的知识点拆解为可逐个攻克的小任务，让学习体验像闯关游戏一样有节奏感和成就感。

#### 5.2.1 词汇冒险（Vocabulary Quest）

| 需求项 | 描述 |
|--------|------|
| 功能目标 | 帮助用户掌握 FCE 要求的约 6000 词汇 |
| 词汇库 | FCE 核心词汇库，按主题分类（环境、科技、教育、社会等） |
| 学习方式 | 多种题型轮换：看图选词、听音拼写、语境填空、同义词配对、造句 |
| 复习机制 | 基于艾宾浩斯遗忘曲线的间隔重复算法（Spaced Repetition） |
| 每日量 | 新词 15–20 个 + 复习词 20–30 个（可调） |
| 游戏化 | 每个主题是一个"区域"，单词是"宝藏"，学完一个主题 = 解锁一个区域 |

**词汇关卡流程：**

```
进入主题区域（如："Environment Island"）
  → 学习新词（每个词展示：发音、释义、例句、搭配）
  → 互动练习（3-5 种题型混合）
  → Boss 战：主题综合测试（全部通过才算征服该区域）
  → 解锁区域奖励 + 下一区域
```

#### 5.2.2 语法挑战（Grammar Dungeon）

| 需求项 | 描述 |
|--------|------|
| 功能目标 | 系统掌握 FCE 考查的全部语法点 |
| 语法库 | 覆盖 FCE 考纲语法：时态体系、条件句、虚拟语气、被动语态、定语从句、倒装句等 |
| 学习方式 | 规则讲解（短视频/图文）→ 例句分析 → 填空 → 改错 → 句型转换 |
| 难度递进 | 每个语法点分 3 级：认知 → 应用 → 综合 |
| 游戏化 | 语法点设计为"地下城关卡"，每个语法点是一个 Boss，击败 = 掌握 |

#### 5.2.3 阅读探索（Reading Expedition）

| 需求项 | 描述 |
|--------|------|
| 功能目标 | 提升阅读速度与理解能力，对标 FCE 阅读题型 |
| 素材库 | 分级阅读材料：短文、新闻、杂志文章、故事等，难度从 B1 到 B2+ |
| 训练题型 | 匹配题、多选题、填空题、标题配对——完全对标 FCE 阅读题型 |
| 阅读辅助 | 内置词典（点击查词）、长难句解析、阅读速度统计 |
| 游戏化 | 每篇文章是一个"探索地图"，读完解锁故事剧情 |

#### 5.2.4 听力训练（Listening Lab）

| 需求项 | 描述 |
|--------|------|
| 功能目标 | 提升听力理解能力，对标 FCE 听力题型 |
| 素材库 | 对话、独白、广播、访谈、演讲等，英式发音为主 |
| 训练题型 | 选择题、填空题、匹配题——对标 FCE 听力题型 |
| 功能特性 | 可调语速（0.75x / 1x / 1.25x）、逐句精听、听写模式、字幕切换 |
| 游戏化 | 听力素材设计为"情报截获"任务，用户扮演特工收集信息 |

#### 5.2.5 写作工坊（Writing Workshop）

| 需求项 | 描述 |
|--------|------|
| 功能目标 | 掌握 FCE 写作要求的两篇文章（议论文 + 体裁写作） |
| 训练内容 | 议论文（Essay）、邮件/信件、评论、报告、故事 |
| 功能特性 | AI 写作批改（语法纠错、词汇建议、结构评分）、范文对照、写作模板 |
| 评分维度 | 对标 FCE 写作评分标准：任务完成度、语言准确性、词汇丰富度、篇章结构、文体适切性 |
| 游戏化 | 写作任务设计为"创作委托"，用户以作者身份完成"约稿"，获得"稿费"奖励 |

#### 5.2.6 口语练习（Speaking Arena）

| 需求项 | 描述 |
|--------|------|
| 功能目标 | 训练 FCE 口语考试的四个环节 |
| 训练内容 | Part 1 个人信息、Part 2 图片描述、Part 3 双人讨论、Part 4 深入讨论 |
| 功能特性 | AI 语音对话（模拟考官提问）、语音识别评分（发音/流利度/语法/词汇）、录音回放对比 |
| 技术要求 | ASR 语音识别 + AI 对话引擎 + 发音评分算法 |
| 游戏化 | 口语练习设计为"竞技场挑战"，用户与 AI 考官"对战"，获得段位提升 |

#### 5.3.2 核心游戏化机制

**① 等级与经验值系统**

| 元素 | 设计 |
|------|------|
| 等级体系 | 1–100 级，对应 FCE 能力成长曲线（Lv.1 零基础 → Lv.100 FCE 卓越） |
| 经验值来源 | 完成每日任务 +50 XP / 完成一个关卡 +100 XP / 模考 +200 XP / 连胜加成 +20% |
| 等级名称 | 1-10 新手冒险者 → 11-25 学徒探索者 → 26-45 资深行者 → 46-70 语言骑士 → 71-90 FCE 战士 → 91-100 语言大师 |
| 视觉反馈 | 升级动画 + 经验条 + 等级徽章 |

**② 连胜打卡系统（Streak）**

| 元素 | 设计 |
|------|------|
| 连胜计数 | 每日完成至少 1 个学习任务即保持连胜，连续天数累计显示 |
| 视觉激励 | 连胜火焰图标 🔥，天数越多火焰越旺盛 |
| 连胜保护 | 提供"连胜护盾"道具（连续 7 天获得 1 个，或金币购买），可在断签时保护连胜 |
| 里程碑奖励 | 7天/30天/100天/365天 连胜解锁特殊徽章和奖励 |
| 心理设计 | 连胜天数越多，用户越不舍得中断——损失厌恶驱动坚持 |

**③ 虚拟货币与商店**

| 货币 | 获取方式 | 用途 |
|------|---------|------|
| 金币 | 完成任务、通关、连胜奖励 | 购买道具、兑换学习素材、解锁特殊关卡 |
| 宝石 | 模考高分、成就解锁、每日签到 | 高级道具、头像装饰、连胜护盾 |

**商店道具示例：**

| 道具 | 效果 | 价格 |

| 连胜护盾 | 自动保护一次断签 | 50 宝石 |
| 时间冻结 | 暂停今日任务截止时间 12 小时 | 30 宝石 |
| 经验药水 | 下次学习获得双倍经验 | 40 金币 |
| 提示卡 | 练习中获得一次提示 | 20 金币 |
| 复活卡 | Boss 战失败后重试（不扣分） | 35 金币 |

**④ 排行榜系统**

| 类型 | 周期 | 分组方式 |
|------|------|---------|
| 周榜 | 每周重置 | 按等级相近分组（每组 30 人），Top 5 升段，末位降段 |
| 月榜 | 每月重置 | 全站排名，展示 Top 100 |
| 好友榜 | 持续 | 好友间对比，突出连胜天数和经验值 |
| 班级榜（Phase 2） | 持续 | 教师/家长可创建班级，查看成员进度 |

**段位设计：**

```
青铜段位 → 白银段位 → 黄金段位 → 铂金段位 → 钻石段位 → 大师段位
（每组 30 人，每周 Top 5 升段，末 5 位降段，制造持续竞争感）
```

### 5.4 FCE 考试中心（P0）

#### 5.4.1 全真模拟考试

| 需求项 | 描述 |
|--------|------|
| 功能目标 | 提供对标真实 FCE 考试的全真模考体验 |
| 模考内容 | 完整模拟 FCE 四个模块：阅读与语言运用（75min）、写作（80min）、听力（40min）、口语（15min） |
| 模考模式 | 严格计时模式（模拟真实考试）+ 自由练习模式（不计时，适合初期） |
| 评分系统 | 客观题自动评分；写作与口语由 AI 评分引擎评分，对标 FCE 评分标准 |
| 成绩报告 | 总分（剑桥量表分）+ 各模块得分 + 与通过线的差距 + 优势/薄弱项分析 |

#### 5.4.2 分项专项训练

针对 FCE 考试每个模块的每种题型提供专项训练：

| 模块 | 专项题型 |
|------|---------|
| 阅读与语言运用 | 多选阅读理解、匹配题、填空题、完形填空、词形转换、开放填空 |
| 写作 | 议论文训练、邮件写作、评论写作、报告写作、故事写作 |
| 听力 | 多选听力、填空听力、匹配听力、选择听力 |
| 口语 | 个人陈述训练、图片描述训练、讨论训练、深入问答训练 |

#### 5.4.3 成绩预测引擎

| 需求项 | 描述 |
|--------|------|
| 功能目标 | 基于用户学习数据预测 FCE 考试成绩 |
| 数据输入 | 日常练习正确率、模考成绩、学习时长、各模块表现 |
| 输出 | 预测分数区间 + 各模块预测分 + 达到通过线的概率 |
| 更新频率 | 每次模考后更新，每周基于日常数据微调 |
| 可视化 | 趋势图展示预测分变化，让用户看到进步轨迹 |

## 6. 用户使用流程

### 6.1 新用户核心流程

```
注册/登录
  │
  ▼
入学能力测评（20-30 分钟）
  │
  ▼
查看测评报告 & 能力分析
  │
  ▼
设置目标（FCE 考试日期 + 每日学习时长）
  │
  ▼
系统生成个性化学习计划
  │
  ▼
进入学习首页 → 开始今日任务
  │
  ▼
完成每日学习任务（词汇/语法/阅读/听力/写作/口语）
  │
  ▼
获得奖励（XP + 金币 + 宝箱）→ 更新连胜
  │
  ▼
查看学习数据与进步反馈
  │
  ▼
（阶段性）参加模考 → 成绩预测更新
  │
  ▼
参加 FCE 考试 → 达成目标 🎉
```

### 6.2 每日学习循环

```
打开网站 → 看到今日任务清单
  │
  ├── 词汇冒险（约 10 分钟）→ 获得金币 + XP
  ├── 语法挑战（约 8 分钟）→ 获得金币 + XP
  ├── 阅读探索（约 10 分钟）→ 获得金币 + XP
  ├── 听力训练（约 8 分钟）→ 获得金币 + XP
  ├── 写作/口语（隔日轮换，约 15 分钟）→ 获得金币 + XP
  │
  ▼
全部完成 → 开启每日宝箱 → 连胜 +1 → 升级判定
  │
  ▼
查看排行榜排名变化 → 预览明日任务 → 退出
```

> **设计要点**：每日任务总量控制在 30–50 分钟（可调），任务间有明确的"完成感"反馈，利用"蔡格尼克效应"（未完成任务的心理张力）让用户期待明天继续。

## 11. 风险与对策
